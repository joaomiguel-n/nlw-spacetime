## 🖥️ Projeto
Esse é um projeto web de uma cápsula do  tempo para exibir memórias em uma linha do tempo.

## 🚀 Tecnologias
Esse projeto foi desenvolvido durante o NLW da RocketSeat com as seguintes tecnologias:

- HTML
- CSS
- Git e Github
